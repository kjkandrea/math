## 순환소수를 분수로 바꾸기

```
0.77777777...
```

1. `x = 0.7...`
2. `10x = 7.7...`
3. `9x = 7`
4. `x = 7/9`

답은 `7/9`

```
1.2222222...
```

1. `x = 1.2...`
2. `10x = 12.2...`
3. `9x = 11`
4. `x = 11/9`

답은 `11/9`

```
0.3636363636...
```

1. `x = 0.3636...`
2. `100x = 36.3636...`
3. `99x = 36`
4. `x = 36/99`
5. `x = 4/11`

답은 `4/11`
